# Product Management System

ระบบ **Product Management System** สำหรับการเรียนรู้และฝึกปฏิบัติเรื่อง **Layered Architecture** ในรายวิชา ENGSE207 (Software Architecture)

โปรเจกต์นี้ออกแบบมาเพื่อใช้ใน **Workshop ก่อนสอบ Midterm** โดยเน้นการแยกความรับผิดชอบของโค้ด (Separation of Concerns) และการ Refactoring จากโค้ดแบบรวมศูนย์ (Monolithic) ไปสู่โครงสร้างแบบเป็นระบบ

---

## วิธีใช้งาน

### 1️. ติดตั้ง Dependencies

```bash
npm install
```

### 2️. รัน Server

```bash
npm start
# หรือ
node server.js
```

### 3️. เปิดใช้งานระบบ

เปิด Browser แล้วเข้า:

```
http://localhost:3000
```
